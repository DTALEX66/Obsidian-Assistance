# V5 Talos Dashboard Demo

这是脱敏 demo，不包含正式 vault 内容。

使用方式：

1. 将 `snippets/v5/talos-dashboard.css` 复制到测试 vault 的 `.obsidian/snippets/`。
2. 在 Obsidian 外观设置中启用该 snippet。
3. 将本目录下 `50_Dashboard/` 和 `40_Wiki/MOC/` 的 Markdown 复制到测试 vault。
4. 打开 `50_Dashboard/今日工作台_Talos.md`。

注意：Dataview 查询在空测试 vault 中可能为空，这是预期行为。
