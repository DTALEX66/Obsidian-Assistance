# Obsidian snippet examples

This folder preserves public-safe TALOS / Purple Gemstone CSS snippet examples without storing them under a tracked `.obsidian/` runtime path.

## Files

- `talos-purple-gemstone-knowledgeos.css` — current TALOS Purple Gemstone KnowledgeOS snippet example.
- `purple-gemstone-console.css` — legacy Purple Gemstone Console snippet example.

## Boundary

These files are reusable design/source artifacts. They are **not** copied from a private vault runtime directory and should not be committed under `.obsidian/snippets/` inside this helper repository.

For real vault installation, copy a reviewed snippet manually into the target vault's `.obsidian/snippets/` directory, enable it in Obsidian Appearance settings, and keep formal-vault changes local-only unless explicitly requested.
