# 组件库与尺寸规范

## 1. 色彩

| Token | 色值 | 用途 |
|---|---:|---|
| `--pg-bg-deep` | `#080B14` | 最深背景 |
| `--pg-bg` | `#0B0E18` | 主背景 |
| `--pg-surface` | `rgba(20,24,42,.78)` | 玻璃卡片 |
| `--pg-accent` | `#8B5CF6` | 主紫色 |
| `--pg-accent-2` | `#A970FF` | 高亮紫 |
| `--pg-blue` | `#38BDF8` | 科技蓝 |
| `--pg-green` | `#78E08F` | 成功/Active |
| `--pg-orange` | `#F59E0B` | 设计/警示 |
| `--pg-red` | `#FF5F7E` | Again/错误 |
| `--pg-gold` | `#F5C96B` | 机会/收藏 |

## 2. 间距

| 场景 | 数值 |
|---|---:|
| 页面外边距 | 16-24px |
| 卡片间距 | 16px |
| 卡片内边距 | 18px / Hero 26px |
| 小按钮 | 8px 12px |
| 大按钮 | 10px 14px |
| 圆角 | 12-16px |

## 3. 组件

### Hero

```html
<div class="pg-hero">
  <p class="pg-kicker">Purple Gemstone Console</p>
  <h1 class="pg-title">页面标题</h1>
  <p class="pg-subtitle">页面说明</p>
</div>
```

### Card

```html
<div class="pg-card">
  <p class="pg-kicker">模块名</p>
  <p>内容</p>
</div>
```

### Pill

```html
<span class="pg-pill">#project</span>
<span class="pg-pill blue">#ai</span>
<span class="pg-pill green">Active</span>
<span class="pg-pill orange">Design</span>
<span class="pg-pill red">Risk</span>
```

### Stat

```html
<div class="pg-stat"><span class="num">89</span><span class="label">Notes Edited</span></div>
```

### Review Button

```html
<div class="pg-review-buttons">
  <div class="pg-review again">Again</div>
  <div class="pg-review hard">Hard</div>
  <div class="pg-review good">Good</div>
  <div class="pg-review easy">Easy</div>
</div>
```

## 4. 组件状态

| 状态 | 设计语言 |
|---|---|
| Normal | 深色玻璃 + 细边框 |
| Hover | 紫色弱高亮 |
| Active | 紫色渐变 + 内发光 |
| Success | 绿色点 / pill |
| Warning | 橙色 pill |
| Error | 红色半透明背景 |
