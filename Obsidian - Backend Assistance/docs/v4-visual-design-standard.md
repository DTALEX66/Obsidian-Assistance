# V4 视觉设计标准

## 目标

让 V4 输出页面呈现学习看板、卡片墙、复习中心的视觉层次，同时不污染整个 Obsidian。

## 设计原则

- 作用域限制在 `.knowledgeos-v4`。
- 支持浅色/深色模式。
- 不依赖外部字体、图片或 URL。
- 兼容默认主题与 Minimal Theme。

## 核心组件

- 课程首页 Hero。
- 状态卡片。
- 知识卡片网格。
- 复习卡片。
- 待审核提示。
- 状态标签。

## 执行要求

CSS 文件放入 `snippets/v4/`，用户可复制到 vault snippets 后手动启用。

## 验收标准

- CSS 无外部 URL。
- 不出现 `body` / `html` 全局选择器。
- 类名与模板 `cssclasses` 对应。

## 安全边界

- 本规范只服务辅助仓库，不要求写入正式 Obsidian vault。
- 文档中不得包含真实课程正文、原始素材、转写全文、OCR 全文、token 或 API key。
- 所有示例路径必须使用占位符或 demo 路径，不能依赖用户正式库路径。
