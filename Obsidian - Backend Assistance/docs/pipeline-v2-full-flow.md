# 课程处理全流程规范

> 版本：v2 — 已跑通第一课全流程
> 适用：从 E 盘课程资料到 Obsidian 知识库的自动转化

---

## 一、流水线总览

```
原始素材 → 素材识别 → 转写/OCR → 核验 → 总结 → 卡片 → 入库报告 → 清理
    1          2           3          4      5      6       7        8
```

| 阶段 | 产出 | 负责 |
|:---|:---|:---|
| 1. 素材识别 | 素材清单 | Hermes Agent |
| 2. 转写/OCR | 音频/视频转写 .txt, PDF OCR .json | 脚本 + Whisper + Tesseract |
| 3. 核验 | 核验报告 | Hermes Agent（从转写中交叉核验） |
| 4. 总结 | 逐节总结 × N | Hermes Agent |
| 5. 卡片 | 知识卡片 + 复习卡片 | Hermes Agent |
| 6. 入库 | 正式课程库写入 | Hermes Agent（出计划 → 等用户确认） |
| 7. 入库报告 | `93_导入报告/` | Hermes Agent |
| 8. 清理 | 95_待审核 归档 | 用户确认后执行 |

## 二、全自动模式（用户只看最终结果）

### 2.1 自动处理原则

- **用户不参与中间环节** — 所有转写、核验、总结、卡片生成由 Agent 自动完成
- **只出最终写入计划** — 写入 Obsidian 前出一份写入计划供用户确认（符合"写入前必须先出计划"要求）
- **入库后只写最终内容** — 中间文件放入 `95_待审核/`，最终内容写入正式库
- **不确定标记 "待人工补充"** — 任何不能从转写/OCR 确认的内容必须标记

### 2.2 核验自动化

第一课验证可行的自动核验方法（优先使用，不依赖联网）：

| 核验类型 | 方法 | 示例 |
|:---|:---|:---|
| 音频转写交叉核验 | 同一内容多份转写对比 | DAY6 Anki 有4个视频转写可交叉核对 |
| PDF OCR 补充 | OCR 提取关键任务页 | DAY20 复盘任务从 PDF OCR 提取 |
| ASR 综合校准 | 上下文 + 课程主线逻辑判断 | "NK/MK" 统一校正为 "Anki" |
| 非核心内容识别 | 按内容类型过滤 | 学员分享、获奖名单、运营信息 → 过滤并写入报告 |

## 三、未来课程过滤规则

### 3.1 应保留的内容

- 课程主线讲解
- 核心知识点、方法、流程
- 老师强调的重点和案例
- 操作步骤/实操流程
- 术语及其解释
- 可转知识卡片/复习卡片的内容

### 3.2 应过滤的内容（不写入正式笔记）

| 类型 | 示例 | 处理 |
|:---|:---|:---|
| 学员个人分享 | 学员 A 的学习经历、心得体会 | 不写入，仅在报告中记录 "有学员分享环节" |
| 获奖名单 | Q&A 大赛获奖者姓名、作品展示 | 不写入，仅在报告中记录 "评选了 N 位优秀学员" |
| 运营信息 | "扫码加小助手微信"、"添加微信xxx" | 不写入 |
| 打卡抽奖信息 | "打卡赢奖品"、"转发有奖" | 不写入 |
| 临时通知 | "课程时间调整"、"直播链接变更" | 不写入 |
| 个人隐私 | 学员真实姓名、微信/电话 | 不写入 |
| 平台推广 | 其他课程广告、折扣信息 | 不写入 |

### 3.3 过滤处理方式

```text
原始课程素材
    ├── 保留 → 进入正式总结/笔记
    ├── 过滤 → 不写入，在导入报告中记录过滤原因
    └── 不确定 → 标记 "待人工补充"
```

**导入报告必须包含：** 过滤了哪些内容 + 为什么过滤

## 四、关键操作规范

### 4.1 写入 E 盘

```
Python 可以直接写 E 盘路径        ✅
Node.js REPL 会报 EPERM           ❌
PowerShell 5.1 不支持 &&          ⚠️ 用分号或 if 替代
PowerShell 5.1 不支持 utf8BOM     ⚠️ 用 `-Encoding UTF8`
```

### 4.2 BOM 问题

百度网盘 YunDetectService 可能为 JSON 注入 BOM，导致 Obsidian 报错。处理方法：
- 杀掉 YunDetectService 进程
- 或重启电脑

### 4.3 Git 备份

- `obsidian-git` 插件：30 分钟自动 commit
- **不推远程**（仅本地备份）
- `.gitignore` 已配置排除：媒体文件、缓存、工作区配置
